# 7. Localisation

> Le contenu du playbook est universel. Ce chapitre documente ce qui change selon le contexte local -- legislation, ecosysteme, outils, pratiques culturelles.

Les chapitres 0 a 6 s'appliquent partout. Ce chapitre est le seul endroit ou le contenu differe par pays : cadre legal (RGPD, CCPA, PIPEDA...), fournisseurs locaux, conventions fiscales et culturelles. Consultez la fiche de votre pays pour adapter votre installation, ou contribuez une nouvelle locale si la votre manque.

Voir `sections/00-reading-guide.md` pour le contrat linguistique complet du playbook.

## Comment ajouter une locale

Ouvrez une Issue avec le label `new-locale` en precisant :
- Le code locale (ex. `de-DE`)
- Un resume de ce qui est specifique localement
- Si vous pouvez rediger le premier brouillon
