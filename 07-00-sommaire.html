Chapter 7 — Localisation

The playbook content is universal (EN-US). This chapter documents what changes per local context — legislation, ecosystem, tools, cultural practices.

Available locales

How to add a locale

Open an Issue with label new-locale and include: - The locale code (e.g. de-DE) - A brief summary of what is locally specific - Whether you can write the first draft

Language contract

Everything in chapters 0–6 is universal. This chapter is the only place where content differs per geography. See sections/00-reading-guide.md for the full language contract.
