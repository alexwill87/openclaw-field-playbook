1.9 -- Est-ce que c'est complique pour moi ?

Vous n'etes pas developpeur. Vous ne connaissez rien a "Docker" ou "SSH". Est-ce que c'est quand meme pour vous ? Oui, et voici pourquoi.

Deux chemins, deux niveaux de difficulte

Chemin 1 : Vous faites installer (recommande)

Difficulte : comme faire installer une alarme chez vous.

Vous appelez un installateur. Il s'occupe de tout. En 3 heures, votre agent est pret. Vous recevez une formation de 30 minutes pour apprendre a utiliser Telegram avec votre agent.

Ce que vous devez savoir faire : - Utiliser Telegram (aussi simple que WhatsApp) - Repondre "ok" ou "non" a un message - Dicter un devis-type une fois pour que l'agent apprenne votre format

C'est tout.

Chemin 2 : Vous installez vous-meme

Difficulte : comme monter un meuble IKEA avec la notice.

Le playbook vous guide etape par etape. Chaque commande est ecrite, chaque resultat attendu est montre. Vous n'avez pas besoin de comprendre ce que font les commandes -- vous les copiez-collez.

Ce que vous devez savoir faire : - Taper des commandes dans un terminal (le guide vous montre comment) - Suivre des instructions dans l'ordre - Lire des messages d'erreur (le guide les explique tous)

Comptez 6-10 heures reparties sur un week-end.

Ce que vous n'avez PAS besoin de savoir

  • Programmer (aucun code a ecrire)
  • Comprendre comment fonctionne l'IA (vous l'utilisez, pas la construisez)
  • Gerer un serveur (une fois installe, ca tourne tout seul)
  • Parler anglais (tout le guide est en francais)

Les moments ou ca peut bloquer

Soyons honnetes : voici les 3 moments ou les gens galèrent le plus.

Moment Pourquoi c'est dur La solution
Choisir un serveur (VPS) Trop de choix, jargon technique Le guide recommande un seul fournisseur avec les bons reglages
Connecter l'email Chaque fournisseur a sa procedure Le guide couvre Gmail, Outlook, et les emails pro
Ecrire les instructions de l'agent Il faut expliquer votre metier a l'agent Le guide fournit des modeles a remplir

Aucun de ces blocages n'est technique. Ce sont des choix a faire, et le guide vous aide a les faire.

Une comparaison simple

Tache du quotidien Difficulte Utiliser OpenClaw
Utiliser WhatsApp Facile Plus facile (vous repondez "ok")
Faire une declaration URSSAF en ligne Moyen Equivalent (suivre un guide)
Installer une box internet Moyen Equivalent (brancher et configurer)
Creer un site internet soi-meme Difficile Plus simple (pas de design a faire)

Le filet de securite

  • L'agent ne fait jamais rien tout seul. Pendant le premier mois, tout passe par votre validation.
  • Vous pouvez tout arreter en un clic. Si ca ne vous plait pas, vous eteignez le serveur. Pas de consequences.
  • Vous n'etes pas seul. La communaute repond aux questions, et un installateur peut prendre le relais a tout moment.

En resume

Si vous savez utiliser WhatsApp et repondre a un email, vous pouvez utiliser OpenClaw. La vraie question n'est pas "est-ce que c'est complique ?", c'est "est-ce que je veux gagner 2 heures par jour ?".
