Politique de confidentialité

Cette page décrit comment le site OpenClaw Field Playbook traite vos données.

Données collectées

Ce site ne collecte aucune donnée personnelle directement.

  • Les commentaires giscus passent par GitHub et sont soumis à la politique de confidentialité de GitHub.
  • Les badges GitHub (étoiles, forks) utilisent l'API publique de GitHub. Aucune donnée utilisateur n'est transmise.

Cookies

Ce site n'utilise aucun cookie.

Le site utilise localStorage pour sauvegarder :

  • Votre préférence de thème (clair ou sombre)
  • Votre progression dans la checklist interactive

Ces données restent uniquement dans votre navigateur. Aucun cookie tiers. Aucun tracking. Aucun outil d'analyse (pas de Google Analytics, pas de Matomo, rien).

Hébergement

Le site est hébergé sur GitHub Pages. Le code source est public et consultable sur GitHub.

GitHub Pages peut collecter des informations techniques (adresse IP, user-agent) dans le cadre de son fonctionnement. Consultez la politique de confidentialité de GitHub pour plus de détails.

Contact

Pour toute question relative à cette politique de confidentialité, vous pouvez ouvrir une issue sur GitHub.
