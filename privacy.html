Politique de confidentialite

Cette page decrit comment le site OpenClaw Field Playbook traite vos donnees.

Donnees collectees

Ce site ne collecte aucune donnee personnelle directement.

  • Les commentaires giscus passent par GitHub et sont soumis a la politique de confidentialite de GitHub.
  • Les badges GitHub (etoiles, forks) utilisent l'API publique de GitHub. Aucune donnee utilisateur n'est transmise.

Cookies

Ce site n'utilise aucun cookie.

Le site utilise localStorage pour sauvegarder :

  • Votre preference de theme (clair ou sombre)
  • Votre progression dans la checklist interactive

Ces donnees restent uniquement dans votre navigateur. Aucun cookie tiers. Aucun tracking. Aucun outil d'analyse (pas de Google Analytics, pas de Matomo, rien).

Hebergement

Le site est heberge sur GitHub Pages. Le code source est public et consultable sur GitHub.

GitHub Pages peut collecter des informations techniques (adresse IP, user-agent) dans le cadre de son fonctionnement. Consultez la politique de confidentialite de GitHub pour plus de details.

Contact

Pour toute question relative a cette politique de confidentialite, vous pouvez ouvrir une issue sur GitHub.
