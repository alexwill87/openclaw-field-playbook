Securite

OpenClaw a connu des incidents de securite significatifs en 2026. Voici ce que vous devez savoir :

IncidentDateImpactMitigation
ClawJackedFevrier 2026Tout site web peut hijacker un agent OpenClaw local via WebSocket. Le rate limiter exempte localhost.Mettre a jour vers la derniere version. Configurer un mot de passe gateway fort.
ClawHavocMars 2026300+ skills malicieuses sur ClawHub avec des noms typosquattes.Verifier le code source. Utiliser le skill Security Auditor. Verifier les scans VirusTotal sur ClawHub.
9 CVE en 4 joursMars 20261 CVE critique (9.9/10), 6 high, 2 medium. 135 000 instances exposees publiquement.Mettre a jour immediatement. Ne JAMAIS exposer le gateway sur Internet public. Utiliser Tailscale.

Outils de securite :

  • NemoClaw (NVIDIA) — Sandbox kernel-level pour les agents. Enterprise preview. En savoir plus
  • KnoxClaw (AccuKnox) — Sandboxing kernel pour les instances OpenClaw. En savoir plus
  • Security Auditor skill — Skill ClawHub qui audite les permissions et les acces de votre agent a runtime.

Recommandation de ce playbook : Ne JAMAIS exposer le port du gateway OpenClaw sur Internet public. Toujours utiliser Tailscale ou un VPN. Toujours verifier le code source d'un skill avant de l'installer. Mettre a jour regulierement.
