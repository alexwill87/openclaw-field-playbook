OpenClaw Field Playbook

Le guide open-source pour installer OpenClaw proprement.
Écrit par un praticien. Testé sur le terrain. Partagé avec tout le monde.

J'ai fait 3 installations ratées avant de comprendre. Ce playbook est le résultat : 92 sections, 7 chapitres, 20 bugs corrigés par un agent terrain. Tout est là pour que vous n'ayez pas à galérer comme moi. — Alex Willemetz

C'est quoi OpenClaw ? Commencer l'installation GitHub

Vous êtes qui ?

Ce guide s'adapte à votre profil. Choisissez le vôtre.

Entrepreneur / Artisan

Vous dirigez une entreprise et vous voulez comprendre ce que l'IA peut faire pour vous — sans jargon.

 CTO / Équipe technique

Vous voulez déployer un agent IA pour votre équipe, sur votre infra, avec vos contraintes.

 Développeur indépendant

Vous savez ce qu'est Docker. Vous voulez les commandes, pas le blabla.

 Agent IA

Vous êtes un agent. Vous clonez ce repo et vous suivez les sections comme un runbook.

Par où commencer

  • Je découvre C'est quoi OpenClaw ? → Définition → Installation
  • J'installe Installation → Configuration → Personnalisation
  • Je veux des exemples Cas d'usage → puis remontez vers les chapitres techniques
  • Je suis un agent IA Lisez CLAUDE.md → Section 2.1 et suivez dans l'ordre
